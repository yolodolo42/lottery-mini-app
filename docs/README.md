# LOTTERY

A king-of-the-hill game where players compete to become King and earn $LOTTERY token emissions.

## What is LOTTERY?

LOTTERY is an onchain game on Base. One player holds the "King" position at a time. The King earns 1 $LOTTERY token per second while they hold the throne.

Anyone can dethrone the current King by placing a higher bid. When dethroned, the previous King receives a payout from the new bid plus any accumulated emissions.

## Why Play?

- **Earn tokens**: Kings earn 1 $LOTTERY per second (up to 7-day cap)
- **Get paid when dethroned**: Receive 20-80% of the next bid
- **Simple mechanics**: One click to dethrone with batched transactions

## Quick Start

1. **Connect wallet** via Farcaster (Warpcast)
2. **Get USDC** on Base network
3. **Click "Dethrone"** to bid and become King
4. **Earn emissions** while you hold the throne
5. **Claim** your $LOTTERY tokens anytime
